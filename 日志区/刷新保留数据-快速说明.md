# 刷新保留数据 - 快速说明 💾

## 功能
刷新浏览器后,当前对局数据会自动保留并恢复。

## 工作原理

### 自动保存 🔄
- 每次游戏状态变化时自动保存到浏览器 localStorage
- 无需手动操作

### 自动恢复 🔄
- 页面加载时自动检测并恢复保存的状态
- 完整恢复所有数据(分数、局数、历史记录等)

### 用户界面 💡

**开始界面 (有保存数据时)**:
```
┌─────────────────────────────────┐
│ 💾 发现保存的进度                │
│ 检测到未完成的对局...            │
│ [继续游戏] [清除进度]            │
└─────────────────────────────────┘
```

**游戏界面**:
```
东风战
东1局 / 0本场 (0立直棒)
💾 自动保存已开启  ← 新增提示
```

## 使用场景

✅ **刷新页面**: F5 刷新,数据不丢失  
✅ **关闭浏览器**: 重新打开,继续游戏  
✅ **意外关闭**: 崩溃重启,数据恢复  
✅ **跨天游戏**: 今天玩,明天继续

## 操作说明

### 继续上次游戏
1. 打开页面
2. 看到"发现保存的进度"提示
3. 点击"继续游戏"按钮 ✅

### 开始新游戏
1. 打开页面
2. 看到"发现保存的进度"提示
3. 点击"清除进度"按钮
4. 确认 → 选择游戏类型 ✅

### 重置当前游戏
1. 游戏中点击"重置"按钮
2. 自动清除保存的数据
3. 回到开始界面 ✅

## 技术细节

**保存位置**: 浏览器 localStorage  
**保存内容**: 完整游戏状态 (玩家、分数、历史等)  
**保存时机**: 每次界面更新时自动保存  
**数据大小**: 约 10-50 KB

## 注意事项

⚠️ 数据仅保存在本地浏览器  
⚠️ 更换浏览器数据不共享  
⚠️ 清除浏览器缓存会删除数据  
⚠️ 多个标签页同时使用可能冲突

## 调试

- ✅ 已恢复上次游戏进度
- 🆕 开始新游戏
- 游戏状态已清除
