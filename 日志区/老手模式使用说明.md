# 日本麻将记录器 - 老手模式使用说明 ⚡

## 功能概述

老手模式是为熟练玩家设计的极速记录模式,无需输入手牌和选择役种,只需直接输入番数和符数即可完成记录。

## 三种分析模式对比

| 模式 | 适用人群 | 输入内容 | 操作复杂度 | 速度 |
|------|---------|---------|-----------|------|
| 🤖 **自动分析** | 初学者 | 13张手牌 + 和牌 | 高 | 慢 |
| ✋ **手动选择** | 中级玩家 | 役种选择 + 符数 | 中 | 中 |
| ⚡ **老手模式** | 高级玩家 | 番数 + 符数 | 低 | 快 |

## 老手模式使用流程

### 步骤 1: 选择基本信息
1. 选择 **胡牌类型** (自摸/点炮/三家责任)
2. 选择 **和牌者**
3. 如果是点炮,选择 **放铳者**
4. 选择 **立直玩家** (如有)

### 步骤 2: 切换到老手模式
在"役种分析模式"区域点击 **⚡ 老手模式** 按钮

### 步骤 3: 输入番数
- **快捷按钮**: 点击 1-13 番的快捷按钮
- **自定义输入**: 在输入框输入 1-99 的任意番数

### 步骤 4: 输入符数
- **快捷按钮**: 点击常用符数 (20, 25, 30, 40, 50, 60, 70, 80, 90, 100, 110)
- **自定义输入**: 在输入框输入 20-110 的符数 (通常为10的倍数)

### 步骤 5: 设置宝牌 (可选)
- **明宝牌**: 设置 dora 数量
- **里宝牌**: 如果有玩家立直,设置 ura-dora 数量
- **赤宝牌**: 设置红五的数量

### 步骤 6: 确认记录
点击 **✅ 确认胡牌** 按钮完成记录

## 番数自动计算说明

老手模式会自动将宝牌番数加入总番数:

```
总番数 = 输入番数 + 明宝牌 + 里宝牌(立直时) + 赤宝牌
```

### 示例

**输入:**
- 番数: 3番
- 符数: 40符
- 明宝牌: 1
- 里宝牌: 2 (玩家立直)
- 赤宝牌: 0

**自动计算:**
- 总番数 = 3 + 1 + 2 + 0 = 6番
- 点数: 按照 6番 40符 计算

## 番数等级提示

在输入区域下方会显示当前番数对应的等级:

| 总番数 | 等级 |
|--------|------|
| 5番 | 满贯 |
| 6-7番 | 跳满 |
| 8-10番 | 倍满 |
| 11-12番 | 三倍满 |
| 13+番 | 役满 |

## 常见使用场景

### 场景 1: 满贯自摸
```
模式: 老手模式
和牌类型: 自摸
番数: 5番
符数: 30符
→ 满贯 12000点 (子家 4000点 × 3)
```

### 场景 2: 跳满点炮
```
模式: 老手模式
和牌类型: 点炮
番数: 6番
符数: 40符
和牌者: 庄家
→ 跳满 18000点
```

### 场景 3: 役满
```
模式: 老手模式
和牌类型: 自摸
番数: 13番
符数: 任意 (役满不计符)
→ 役满 48000点 (庄家) 或 32000点 (子家)
```

## 注意事项

### ⚠️ 必须设置项
- 和牌者
- 番数 (至少1番)
- 符数 (默认30符)

### 💡 使用技巧
1. **快速记录**: 使用快捷按钮比输入框更快
2. **宝牌处理**: 输入番数时不包含宝牌,系统会自动添加
3. **役满简化**: 役满时符数任意,不影响点数
4. **立直里宝**: 只有立直玩家才会计算里宝牌

### 🔍 验证提示
- 番数不能为 0
- 符数最小 20 符
- 必须选择和牌者
- 点炮时必须选择放铳者

## 与其他模式的切换

### 从自动分析切换到老手模式
- 手牌输入会被隐藏
- 清空已输入的手牌
- 保留立直状态和宝牌设置

### 从手动选择切换到老手模式
- 役种选择会被隐藏
- 清空已选择的役种
- 符数设置会重置为 30 符

### 从老手模式切换到其他模式
- 番数和符数会被重置
- 恢复对应模式的输入界面

## 适用场景

**推荐使用老手模式的情况:**
- ✅ 熟悉番符计算规则
- ✅ 需要快速记录多局游戏
- ✅ 已经手动计算过番符
- ✅ 使用麻将计算器得到了番符数据

**不推荐使用老手模式的情况:**
- ❌ 不确定番符计算结果
- ❌ 需要系统帮助分析役种
- ❌ 学习阶段需要验证理解
- ❌ 复杂役种组合需要系统确认

## 总结

老手模式是为**熟练玩家**设计的**极速记录工具**,通过直接输入番数和符数,跳过繁琐的手牌和役种输入,实现最快速度的游戏记录。适合已经掌握番符计算的高级玩家使用。
